n8n wordpress to twitter to mastodon

Nov 11, 2024
1 minute read

ifttt

if then then then 大概这个意思吧。2015年左右用过 当时就是同步wp到twitter weibo facebook 都可以的。 记得当时非常简单登陆 connect账户就行 之前免费。现在是pro会员才行 每月2.9刀费用，对于美刀收入2.9，就当税了，或者小费. 国内的话99就可以搞个一年的特价虚拟空间主机了。

n8n

n8n有自建服务。 之前serv00免费空间注册上了。 也部署上了 n8n serv00部署 问题 – ACEVS 教程我看了几遍 。还是 没搞明白。 从原始需求找到适合自己的节点 | 简单易懂的现代魔法 wp发个post同步twitter 和mastodon 搜索，ai问了问，大致思路有了。wordpress 登陆 get all post 然后筛选 留下最新的post 然后输出 json tillt 和url 思路清晰了。发现操作起来都不懂。 没有直接的模版，只有去看看其他模版了。 有个wordpress to medium的模版，http request 抓取html信息。然后筛选输出想要的信息。我要的就是title 和 url 然后twitter和mastodon发布信息。 twitter授权还得申请开发者模式。还好不难。 mastodon n8n上搜索没有，去youtube搜索教程。详细仔细，照着做就行。设置好 得等几分钟。有延迟估计，我以为失败了，后来发现成功了。 定时执行 加上schedule trigger 我设置每天下午6.00执行一次 。 test workflow 无数次 昨天我看自动运行成功。
